🚀 Step 1: Create Custom Object (Student)

Go to: Setup → Object Manager → Create → Custom Object

Fill details:
Label: Student
Object Name: Student
Record Name: Student Name (Text)
➕ Add Fields

Go to Fields & Relationships → New

Create these:

Field Label	Type	Notes
Roll No	Number	Unique if possible
Class	Text	e.g. FE, SE
Mobile No	Phone	Better than text
🚀 Step 2: Enable Tabs

Go to: Tabs → New → Custom Object Tab

Select: Student
Choose any icon
Add to apps
🚀 Step 3: Apex Controller

Go to: Developer Console → File → New → Apex Class
